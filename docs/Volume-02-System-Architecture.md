# Purpose

This document defines the overall architecture of the SyLinQ platform.

It describes how every major component works together to create one secure, scalable, AI-powered social platform.

---

# Overall Architecture

SyLinQ consists of six primary systems:

1. Frontend
2. Backend
3. Database
4. AI Platform (T1GOX)
5. Storage System
6. External Services
